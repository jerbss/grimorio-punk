## Sobre o Projeto

Uma landing page para apresentar 5 arcanos de investigação criativa, todos desenvolvidos em **HTML, CSS e JavaScript puros**.

### Os 5 Arcanos

1. **I. O Zine de Aristóteles** - Reflexão sobre ética e tarô
2. **II. O Medidor de Caos** - Mistura de zines (Tarô + Política + Pokémon)
3. **III. O Pentáculo dos Limites** - Mapa mental com bloqueios criativos
4. **IV. A Cidade de Papel** - Papercity com lore inusitada
5. **V. O Receptáculo** - Caixa de fósforos como zine-brinquedo

## Direção de Arte

**Estilo:** Pergaminho Místico (Light Academia Esotérica)
- Fundo: Creme claro com texturas finas
- Tipografia: Cinzel + Cormorant Garamond
- Molduras: Linhas simples e respeitosas
- Filosofia: Deixar as imagens em papel branco serem protagonistas

## Tecnologias

- **HTML5** - Semântica pura
- **CSS3** - Grid, Flexbox, Gradientes
- **JavaScript Vanilla** - Scroll suave, navegação interativa
- **Google Fonts** - Tipografia elegante

## Como Usar

1. Abra `index.html` no navegador
2. Adicione suas imagens em `assets/img/`
3. Edite o conteúdo das seções de projeto conforme necessário
4. Use `testes/teste-opcoes.html` para comparar estéticas (se precisar revisar)
